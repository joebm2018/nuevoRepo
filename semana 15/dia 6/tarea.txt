HACER UN SISTEMA DE CREACIÓN DE USUARIOS EN CONSOLA 
CONECTADO A UNA BASE DE DATOS.
EL SISTEMA DEBE PERMITIR HACER UN LOGIN, SI NO SE ESTÁ 
CORRECTAMENTE LOGUEADO, NO DEBE PERMITIR MÁS ACCIONES.
TENEMOS UN OPCIÓN PARA CREAR USUARIOS INVITADOS.
UNA VEZ DENTRO DEL SISTEMA (LOGIN EXITOSO) DEBE PERMITIR 
LAS SIGUIENTES ACCIONES:

SI EL USUARIO ES ADMINISTRADOR:

CREACIÓN DE USUARIOS (SOLO PUEDE CREAR ADMINISTRADORES)
BORRADO DE USUARIOS (NO DEBE PERMITIR BORRARSE A SÍ MISMO)
CAMBIO DE CONTRASEÑA (DEL USUARIO QUE HA INGRESADO)
LISTAR A TODOS LOS USUARIOS
LOGOUT (CERRAR SESIÓN)

SI EL USUARIO ES INVITADO:

CAMBIAR DE CONTRASEÑA
LISTAR A LOS USUARIOS INVITADOS
BORRAR SU PROPIA CUENTA
LOGOUT

TABLA USUARIOS
COLUMNAS:
CORREO,	CONTRASEÑA, NOMBRE, FECHA DE NACIMIENTO, PRIVILEGIOS
